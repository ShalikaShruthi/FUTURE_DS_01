1️⃣ Business Sales Dashboard from E Commerce Data (Future Interns Data Science and Analytics Internship Task 1)

Overview

This project focuses on analyzing e-commerce sales data to identify the best-selling products, sales trends and high revenue regions and categories. Uses python for data cleaning and then PowerBI to build an interactive dashboard that supports data driven decision-making.

Features

1. Cleans the data.
2. Identifies the best selling and most profitable products.
3. Analyses patterns like monthly sales trends, category-wise performance and customer behaviour.
4. Presents insights through a dashboard using PowerBI.

Project Files

superstore.xls
cleaningupdata.py
States vs. Sales.xls
Business Dashboard.pbix

Dashboard Features

1. KPI cards (for quick display of key business metrics)
2. Visualzations (analyzed patterns represented in charts)
3. Interactivity (Filters and slicers)

How to Use this Project

1. Run the Python script to clean the CSV data
2. Review the cleaned and aggregated datasets
3. Open the Power BI file (.pbix) to explore the dashboard
4. Use filters and visuals to analyze business performance
